# Security Policy

## Supported Version

Security fixes are applied to the latest version on the default branch.

## Reporting a Vulnerability

Use GitHub's private security advisory feature for the repository. Include the affected behavior, reproduction steps, impact, and any suggested mitigation.

If private advisories are unavailable, open a public issue requesting a private contact channel without disclosing exploit details.

Please allow maintainers reasonable time to investigate and publish a fix before public disclosure.
