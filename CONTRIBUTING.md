# Contributing

Thank you for helping improve Elementra.

## Development

1. Fork the repository and create a focused branch.
2. Install the locked dependencies with `npm ci`.
3. Make the smallest coherent change that solves the issue.
4. Add or update tests for behavior changes.
5. Run `npm run check`.
6. Open a pull request describing the problem, solution, and verification.

Keep accessibility, keyboard behavior, mobile layouts, and both color themes in scope for UI changes. Avoid adding runtime dependencies when a small local implementation is sufficient.

## Bug Reports

Include reproduction steps, expected and actual behavior, browser and operating system versions, and screenshots when the issue is visual. Do not include sensitive vulnerability details in public issues.
